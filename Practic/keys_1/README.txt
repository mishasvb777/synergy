Описание объектных моделей кода
Python:
    Функция process_array принимает два аргумента: массив A и число B.
    Переменные sum_positive, count_positive, count_greater_B, product_greater_B используются для хранения промежуточных результатов.
    Цикл for перебирает элементы массива, выполняя проверки на положительность и сравнение с B.
    Условие if not has_greater_B обрабатывает случай, когда нет элементов, больших B.
    
JavaScript:
    Функция processArray имеет аналогичную структуру.
    Переменные объявлены с использованием let.
    Цикл for...of перебирает элементы массива.
    Логика работы идентична Python-версии.
